Chess Engine

A solo project — a fully functional chess engine in Java with complete rule enforcement (castling, en passant, pawn promotion). Implemented Alpha-Beta Pruning to cut evaluated game-tree nodes by ~50% vs. standard minimax, improving move latency at depth 7. Integrated MySQL for game history and post-match replay.
